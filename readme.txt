Вы видите перед собой исходный код будущего редактора .class-файлов. Возможно, он пригодится Вам... Код свободный. Вы можете использовать его в ваших проектах (за исключением, конечно, наглого присвоения 1-в-1 (-: ).
На данный момент, более-менее устойчиво работают функции пакета jbedit.classFileStructure. Работа остальных пакетов не гарантируется.
Пакет jbedit.classFileStructure содержит следующие вещи:
jbedit.classFileStructure.ClassFile - основной тип =)
jbedit.classFileStructure.ClassFileLoader - загрузчик класса
jbedit.classFileStructure.ClassFileSaver - сохраняет класс
